# 📝 Todo App (React)

A simple and beautiful Todo List App built using React.
This app allows users to add, update, delete, and mark tasks as completed.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✏️ Update tasks (convert to uppercase)
* ✅ Mark tasks as done / undone
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* React (useState)
* JavaScript (ES6)
* HTML & CSS

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/todo-react-app.git
```

2. Navigate to project folder:

```
cd todo-react-app
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm run dev
```
